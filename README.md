# SmartBank
# 💳 FinSight AI — Smart Banking System with AI Analytics

> 🚀 An intelligent command-line banking system powered by Python that tracks transactions, analyzes financial behavior, and generates insightful reports with visualizations.

---

## 🌟 Overview

**FinSight AI** is a smart banking assistant that goes beyond basic deposit and withdrawal operations. It integrates AI-like features such as:

- 📊 Financial analytics  
- 📅 Monthly reporting  
- 📍 Transaction tracking (time & location)  
- 📈 Data visualization  

This project is ideal for learning **Python, data handling, and basic financial analytics**.

---

## ✨ Features

### 💰 Core Banking Functions
- Check account balance  
- Deposit money  
- Withdraw money (with validation)  

### 🤖 AI-Powered Features
- Automatic transaction logging  
- Smart tagging with:
  - 🕒 Timestamp (date & time)
  - 📍 Location (simulated AI-based detection)

### 📊 Analytics Dashboard
- Total deposits & withdrawals  
- Net cash flow  
- Transaction count  
- Detailed transaction history  

### 📅 Monthly Reports
- Monthly total deposits  
- Monthly total withdrawals  
- Net monthly savings  
- Trend visualization  

### 📈 Data Visualization
- Bar chart for transaction history  
- Line chart for monthly trends  

---

## 🛠️ Tech Stack

- **Language:** Python 🐍  
- **Libraries:**
  - matplotlib → Data visualization  
  - datetime → Timestamp tracking  
  - random → Simulated AI location  
  - collections → Data grouping
